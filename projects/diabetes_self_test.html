Diabetes Test

As someone with a strong family history of diabetes and a deep interest in biological sciences—which I've explored extensively in my genetics lab work—I felt compelled to create a project that could combine my technical skills with a meaningful application. This led to the development of a diabetes prediction web application designed to assist both patients and doctors in assessing the likelihood of diabetes based on user-provided features.

Overview of the Project

The web app allows users to input various health metrics, including BMI, age, weight, and the number of pregnancies. These inputs are processed through a machine learning model to predict whether an individual is likely to have diabetes. The tool is built to be user-friendly and accessible, aiming to provide a preliminary risk assessment.

Data and Preparation

To build this application, I started with a dataset containing 767 entries of female individuals, each with features such as BMI, age, weight, and a diabetes diagnosis label. The dataset served as the foundation for the machine learning model. After importing essential libraries like Scikit-learn, NumPy, and Pandas, I performed feature engineering to refine the dataset. This involved removing unnecessary labels and standardizing the data to a range of -1 to 1 to improve model accuracy. The data was then split into training and test datasets for validation.

Machine Learning Model

For the prediction task, I implemented a Support Vector Machine (SVM) classifier. The model was trained on the prepared data to classify new inputs as either diabetic or non-diabetic. After training, I evaluated the model's performance, achieving an accuracy of 78% on the test data.

Deployment

To make the application accessible, I used Streamlit to develop a web-based interface and deployed it online. The web app allows users to input their data and receive instant predictions, making it a practical tool for preliminary diabetes risk assessment.

Conclusion

This project is a reflection of my passion for integrating data science with biology to address real-world challenges. It represents a meaningful step in my journey to combine technical expertise with biological research to make impactful contributions to healthcare.

Feel free to explore the application and learn more about how it works!
